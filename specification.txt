
Tagging
	The "#" character can be used to tag a task.
	A number of special tags are defined (in place of options):
		Status: #done, #failed, #partial
		Periodic: #everyday, #weekday, #weekend, #monday, #tuesday, ...
		Date: #date=yyyy-mm-dd, #deadline=yyyy-mm-dd
		Misc: #hidden, #essential

Interface
	Command Line
		The data-file must be human readable. Default = "todolist.txt"
		Usage: todo [list|add|edit|delete] [<date>|periodic]
		ASCII Table Formatting: easy_install prettytable
	Web browser # incomplete
		Types
			A single page that contains all the necessary information.
			A Real-Time updated page from when changes also can be made.
				Implementation: ajax/http-rest, file-watch
		Implementation
			jQuery, Bootstrap
